I'm a computer science student and AI researcher focused on **Natural Language Processing** and **Graph Algorithms**. I love building systems that help machines understand human language — from plagiarism detection to semantic clustering.

- 🔭 Currently working on: **Improving plagiarism detection using graph-based NLP**
